# Astrologer CRM

Astrologer CRM is a web-based customer relationship management system built to simplify the management of astrology consultation services. The platform enables administrators to manage astrologers, schedule consultations, track appointments, monitor business metrics, and maintain organized client records through a centralized dashboard.

## Features

* Dashboard with key business statistics
* Astrologer management
* Consultation scheduling and tracking
* Appointment status management
* Client information management
* Search and filtering functionality
* Responsive user interface

## Tech Stack

* HTML
* CSS
* JavaScript

## Future Enhancements

* User authentication
* Database integration
* Payment gateway support
* Video consultation support
* AI-powered consultation summaries
* Notification and reminder system
