# AI Usage Declaration

## Overview

Artificial Intelligence (AI) tools were used during the development of this project to assist with planning, code generation, debugging, and documentation.

## AI Tools Used

* ChatGPT (OpenAI)

## How AI Was Used

The AI tool was used for:

* Generating initial project ideas and structure
* Assisting with frontend UI development
* Providing sample code snippets
* Debugging and resolving implementation issues
* Creating documentation files such as README.md and PROJECT_NOTES.md
* Suggesting improvements and future enhancements

## Developer Responsibility

All AI-generated content was reviewed, modified, tested, and integrated manually. The final project structure, implementation decisions, testing, and submission were completed by the developer.

## Limitations

AI assistance was used as a productivity tool and not as a replacement for software engineering judgment. Any generated code was verified before inclusion in the final project.

## Declaration

I acknowledge that AI tools were used to support the development process. I take full responsibility for the final submitted work, including its functionality, accuracy, and design decisions.
